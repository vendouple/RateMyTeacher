# RateMyTeacher - Feature Plan

Feature: Application for Teacher (MVP)

Tech stack:

- ASP.NET Core 9 MVC
- SQLite (embedded)
- Entity Framework Core 9
- Google Gemini 2.5 Flash via Google Generative AI
- Client: Razor Views, vanilla JS, Chart.js

Structure:

- Controllers, Models, Views, Services, Data
- Resources for localization

Deliverables:

- Minimal MVP: teacher listing, rating, leaderboard, AI summary generation
- .env for Gemini API key
- Embedded SQLite DB
